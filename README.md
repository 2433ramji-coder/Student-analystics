# Student Dashboard Application

A modern, responsive dashboard application for students and administrators with authentication, charts, and data management.

## Features

### Student Features
- View academic scores with charts and tables
- Check attendance records and trends
- Download academic reports
- View overall performance statistics

### Admin Features
- Update student scores
- Mark attendance
- Generate various types of reports
- View system statistics

### Common Features
- User authentication (Login/Logout)
- Protected routes based on user role
- Responsive design for all devices
- Modern UI with charts and visualizations
- Session persistence using localStorage

## Tech Stack

- **React** - Frontend library
- **React Router DOM** - Routing
- **Chart.js & react-chartjs-2** - Data visualization
- **Context API** - State management for authentication
- **CSS3** - Styling with modern features
